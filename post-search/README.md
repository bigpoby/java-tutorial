# TXT 파일 검색 및 단어 검색 프로그램 v2.0 (한글 지원)

윈도우 디렉토리를 입력받아 txt 파일의 목록을 출력하고, 특정 단어를 검색하여 결과를 파일로 저장하는 자바 프로그램입니다. **한글 검색을 완전히 지원**합니다.

## 프로그램 구성

### 1. TxtFileFinder.java
- 기본 버전: 지정된 디렉토리에서만 txt 파일을 검색
- 파일 이름, 크기, 수정일 정보 표시

### 2. TxtFileFinderRecursive.java
- 고급 버전: 하위 디렉토리까지 재귀적으로 검색 가능
- 사용자가 재귀 검색 여부를 선택할 수 있음
- 전체 경로와 파일 정보 표시

### 3. TxtWordSearcher.java ⭐ 
- **통합 버전**: txt 파일 검색 + 단어 검색 + 자동 저장 기능
- 특정 단어가 포함된 행을 찾아 화면 출력 및 자동 파일 저장
- 검색 결과에 파일명, 행 번호, 행 내용 포함
- 성능 모니터링 및 통계 제공

### 4. 핵심 유틸리티 클래스들
- **FileManager.java**: 파일 시스템 작업 및 디렉토리 검증
- **SearchEngine.java**: 텍스트 검색 엔진 (한글 지원)
- **ResultDisplayer.java**: 결과 출력 및 UI 처리
- **ResultSaver.java**: 검색 결과 파일 저장
- **PerformanceMonitor.java**: 성능 측정 및 통계

### 🚀 **실행 방법**

#### **Option 1: 기본 파일 검색** 
```powershell
java TxtFileFinder
```
- 지정한 디렉토리에서 txt 파일만 검색
- 파일명, 크기, 수정일 정보 표시

#### **Option 2: 재귀 파일 검색**
```powershell
java TxtFileFinderRecursive
```
- 현재 디렉토리만 또는 하위 디렉토리까지 선택 가능
- 전체 경로와 파일 정보 표시

#### **Option 3: 통합 단어 검색** ⭐ **추천**
```powershell
java TxtWordSearcher
```
- 파일 검색 + 단어 검색 + 자동 저장 + 성능 측정
- 한글 검색 완벽 지원
- 가장 완전한 기능 제공

#### **1단계: 프로그램 실행**

#### **4단계: 파일 목록 확인**

```
검색 방식: 재귀 검색 (하위 디렉토리 포함)
발견된 txt 파일 개수: 15
```

#### **5단계: 검색어 입력 (한글 지원)**

**한글 검색 예시:**
- 한글 단어: `한글`, `테스트`, `프로그램`, `검색`
- 영문 단어: `test`, `function`, `search`
- 한영 혼용: `한글test`, `program테스트`

#### **6단계: 결과 확인**
```
=== 단어 검색 결과 ===
검색어: 한글검색
총 발견된 항목: 6
검색 시간: 45.32 ms
검색된 파일 수: 15개

파일: korean_test.txt
----------------------------------------
행 1: 안녕하세요. 이것은 한글 테스트 파일입니다.
행 3: 검색 기능이 한글을 제대로 처리하는지 확인합니다.
```

#### **7단계: 결과 자동 저장**
```
=== 저장 완료 ===
저장된 파일: output/한글검색_20250724_141907.txt
저장된 항목 수: 6
```
**자동 저장 규칙:**
- 파일명 형식: `검색어_YYYYMMDD_HHmmss.txt`
- 저장 위치: `output/` 폴더
- 검색 결과가 있을 때만 저장

#### **8단계: 성능 통계 확인 (선택)**
```

=== 성능 통계 ===
전체 실행 시간: 1,234.56 ms
파일 검색 시간: 12.34 ms
단어 검색 시간: 45.32 ms
검색된 파일 수: 15개
```

### 💡 **사용 팁**

#### **효율적인 검색을 위한 팁**:
1. **디렉토리 선택**: 
   - 불필요한 시스템 디렉토리는 피하세요
   - 검색 범위를 적절히 제한하세요

2. **검색어 입력**:
   - **한글 검색 완벽 지원**: 한국어 단어나 문장 검색 가능
   - 대소문자 구분 없음 (영문의 경우)
   - 부분 단어 검색 가능
   - 공백 포함 검색어도 지원
   - **한영 혼용 검색어** 지원

3. **결과 파일**:
   - **자동 저장**: 검색 결과가 있으면 무조건 저장
   - **파일명 형식**: `검색어_YYYYMMDD_HHmmss.txt`
   - **저장 위치**: `output/` 폴더
   - UTF-8 인코딩으로 저장

#### **성능 최적화**:
- 대용량 디렉토리 검색 시 재귀 검색 신중히 선택
- 메모리 사용량 모니터링 활용
- 처리 속도 정보로 시스템 성능 파악

### ⚠️ **주의사항**

1. **권한**: 접근 권한이 없는 디렉토리는 검색되지 않습니다
2. **파일 크기**: 매우 큰 파일의 경우 처리 시간이 오래 걸릴 수 있습니다  
3. **인코딩**: UTF-8 인코딩을 사용하므로 다른 인코딩 파일은 올바르게 표시되지 않을 수 있습니다
4. **메모리**: 대량의 파일 처리 시 메모리 사용량을 확인하세요

## 주요 기능

### 📁 파일 검색 기능
- 디렉토리 유효성 검사
- txt 파일 확장자 검사 (대소문자 구분 없음)
- 파일 정보 표시 (크기, 수정일)
- 재귀적 하위 디렉토리 검색

### 🔍 단어 검색 기능 (NEW!)
- **한글 검색 완전 지원**: 한국어 텍스트 검색 가능
- 파일 내용에서 특정 단어 검색 (대소문자 구분 없음)
- **한영 혼용 텍스트** 검색 지원
- 검색 결과에 파일명, 행 번호, 행 내용 표시
- 검색 결과를 텍스트 파일로 저장
- UTF-8 인코딩 지원으로 완벽한 한글 처리

### 🎨 사용자 인터페이스
- 오류 처리 및 사용자 친화적 메시지
- 단계별 안내 및 선택 옵션 제공
- 결과 저장 여부 및 파일명 설정
